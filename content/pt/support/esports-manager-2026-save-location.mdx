<Section icon={Shield} title="Passo a Passo: Fazendo Backup dos seus Arquivos de Salvamento">

Os backups manuais são essenciais no **esports manager 2026** porque o jogo apresenta uma lógica de simulação complexa. Um único torneio ruim no Apex Masters ou uma negociação fracassada com uma superestrela como Simple pode arruinar meses de progresso. Seguir uma rotina rígida de backup garante que você possa "voltar no tempo" se suas decisões estratégicas derem errado.

<Steps>
  <Step num="1" title="Feche o Jogo">
    Certifique-se de que o jogo e o Steam estejam completamente fechados para evitar o bloqueio de arquivos ou gravações parciais no **local de salvamento do esports manager 2026**.
  </Step>
  <Step num="2" title="Navegue até o Diretório de Salvamento">
    Use o caminho `%LOCALAPPDATA%/EsportsManager2026/Saved/SaveGames` para encontrar seus arquivos `.sav` atuais.
  </Step>
  <Step num="3" title="Copie para uma Pasta Externa">
    Crie uma pasta chamada "EM2026_Backups" na sua área de trabalho ou em uma unidade na nuvem (Google Drive/Dropbox). Copie toda a pasta `SaveGames` para lá.
  </Step>
  <Step num="4" title="Renomeie para Facilitar a Identificação">
    Renomeie a pasta copiada com a data atual e o nome da sua organização (por exemplo, `2026-07-15_NIP_PostDeviceSigning`).
  </Step>
</Steps>

<Callout type="warning" title="Aviso">
Nunca modifique os nomes dos arquivos `.sav` diretamente dentro da pasta do jogo. Isso pode causar incompatibilidade no banco de dados interno, levando a erros de "Salvamento Corrompido" quando você tentar carregar o perfil de manager.
</Callout>

</Section>
