<Section icon={Shield} title="Passos para Instalação Manual de Bancos de Dados Personalizados">

Se você não estiver usando o Steam Workshop, precisará realizar uma instalação manual. Isso é comum para mods de "Conversão Total" que alteram mais do que apenas os nomes dos jogadores. Seguir o caminho correto do diretório é crítico para que o jogo reconheça os novos dados.

<Steps>
  <Step num="1" title="Baixar e Extrair">
    Baixe o arquivo ZIP ou RAR do banco de dados de uma fonte confiável. Use uma ferramenta como 7-Zip ou WinRAR para extrair o conteúdo para uma pasta temporária na sua área de trabalho.
  </Step>
  <Step num="2" title="Localizar o Diretório do Jogo">
    Navegue até os arquivos locais do seu jogo. Para a maioria dos usuários, o caminho é: `C:\Usuários\[SeuNome]\Documentos\Esports Manager 2026\Databases`. Se a pasta não existir, crie-a manualmente.
  </Step>
  <Step num="3" title="Transferir Arquivos">
    Copie a pasta extraída (contendo os arquivos `.db` ou `.json`) para o diretório `Databases`. Certifique-se de não ter pastas aninhadas (por exemplo, `Databases/NomeDoMod/NomeDoMod/arquivos`).
  </Step>
  <Step num="4" title="Verificar a Estrutura dos Arquivos">
    Abra a pasta recém-movida. Você deve ver um arquivo chamado `config.json` ou `meta.data`. Este arquivo diz ao jogo como carregar os recursos específicos.
  </Step>
</Steps>

| Passo da Instalação | Erro Comum | Solução |
| :--- | :--- | :--- |
| **Extração** | Arquivo ZIP corrompido | Baixe novamente com uma conexão estável |
| **Caminho** | Pasta errada selecionada | Use o caminho dos Documentos, não do Arquivos de Programas |
| **Nomenclatura** | Caracteres especiais na pasta | Renomeie a pasta para um texto simples em inglês |
| **Permissões** | Acesso à pasta apenas para leitura | Execute o seu explorador de arquivos como Administrador |

<Callout type="success" title="Sucesso">
Assim que os arquivos estiverem no diretório correto, eles aparecerão no menu suspenso "Seleção de Banco de Dados" quando você clicar em "Novo Jogo" no menu principal.
</Callout>

</Section>

<Section icon={Sparkles} title="Ativando o Banco de Dados no Jogo">

Apenas colocar os arquivos na pasta não é suficiente; você deve instruir explicitamente o **Esports Manager 2026** a usar os dados personalizados em vez do elenco padrão de 2026.

Ao iniciar o jogo, siga estas configurações específicas para garantir que o banco de dados personalizado seja inicializado corretamente:

1. Clique em **Nova Carreira**.
2. Na tela de **Seleção de Banco de Dados**, clique no menu suspenso.
3. Selecione o nome do seu banco de dados personalizado (ex: "Global Realism Pack v1.2").
4. Aguarde a barra de progresso de "Validação de Dados" ser concluída.
5. Selecione as ligas que deseja carregar.

| Recurso | Banco de Dados Padrão | Banco de Dados Personalizado |
| :--- | :--- | :--- |
| **Nomes dos Jogadores** | Fictícios/Aleatórios | Pro-players Profissionais Reais |
| **Logotipos das Equipes** | Formas Genéricas | Identidade Visual Oficial das Equipes |
| **Premiações de Torneios** | Estimadas | Valores Reais do Mundo Real de 2026 |
| **Nomes de Patrocinadores** | Marcas Inventadas | Marcas Globais de Tecnologia/Energia |
| **Atributos da Comissão** | Equilibrados/Padrão | Baseados no Desempenho Real |

<Callout type="info" title="Nota Importante">
Alguns bancos de dados são "Grandes" e podem aumentar o tempo de carregamento inicial do seu jogo. Se o jogo parecer travado em 88%, dê a ele alguns minutos para processar os milhares de novos registros de jogadores.
</Callout>

</Section>

| Tipo de Recurso | Nome da Pasta | Extensão do Arquivo |
| :--- | :--- | :--- |

| **Skins/Interface** | `Interface` | .xml |
