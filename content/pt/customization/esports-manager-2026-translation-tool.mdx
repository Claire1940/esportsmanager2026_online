  <Step num="1" title="Baixe os Arquivos">
    Obtenha os arquivos de localização e bancos de dados de nomes reais mais recentes em hubs de comunidade verificados. Certifique-se de que os arquivos correspondam à versão atual do jogo.

  <Step num="2" title="Localize o Diretório">
    Navegue até a pasta de instalação do jogo. Localize as pastas designadas para arquivos de localização e bancos de dados personalizados.
  </Step>
  <Step num="3" title="Aplique os Arquivos">
    Extraia e coloque os arquivos de tradução na pasta de localização. Copie os arquivos do banco de dados personalizado para o diretório principal do banco de dados.
  </Step>
  <Step num="4" title="Inicie e Verifique">
    Abra o jogo, vá em configurações e selecione seu idioma preferido. Inicie uma nova carreira para verificar se os elencos do mundo real foram carregados corretamente.
  </Step>
