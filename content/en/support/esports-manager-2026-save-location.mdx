<Section icon={Shield} title="Step-by-Step: Backing Up Your Save Files">

Manual backups are essential in **esports manager 2026** because the game features complex simulation logic. A single bad tournament in the Apex Masters or a failed negotiation with a superstar like Simple can derail months of progress. Following a strict backup routine ensures you can "roll back the years" if your strategic decisions backfire.

<Steps>
  <Step num="1" title="Close the Game">
    Ensure the game and Steam are completely closed to prevent file locking or partial writes to the **esports manager 2026 save location**.
  </Step>
  <Step num="2" title="Navigate to the Save Directory">
    Use the path `%LOCALAPPDATA%/EsportsManager2026/Saved/SaveGames` to find your current `.sav` files.
  </Step>
  <Step num="3" title="Copy to External Folder">
    Create a folder named "EM2026_Backups" on your desktop or a cloud drive (Google Drive/Dropbox). Copy the entire `SaveGames` folder there.
  </Step>
  <Step num="4" title="Rename for Clarity">
    Rename the copied folder with the current date and your organization name (e.g., `2026-07-15_NIP_PostDeviceSigning`).
  </Step>
</Steps>

<Callout type="warning" title="Warning">
Never modify the `.sav` file names directly within the game folder. This can cause the internal database to mismatch, leading to "Corrupted Save" errors when you try to load the manager profile.
</Callout>

</Section>
