<Section icon={Map} title="Step-by-Step Installation Guide">

To get the game running on your Mac, you need to set up a compatibility environment. Using Whisky is one of the most efficient, open-source methods available for macOS users. It utilizes Apple's Game Porting Toolkit to translate DirectX calls into Metal, allowing the game to run smoothly.

Follow these steps to configure your environment and launch the game.

<Steps>
  <Step num="1" title="Download and Install Whisky">
    Download the latest version of Whisky for macOS. Drag the application to your Applications folder and open it to complete the initial setup.
  </Step>
  <Step num="2" title="Create a New Bottle">
    Click the create bottle button. Select Windows 10 as the operating system version and name the bottle. This environment will house your Steam installation.
  </Step>
  <Step num="3" title="Install Steam inside the Bottle">
    Download the Windows Steam installer (.msi or .exe). In Whisky, select your bottle, click Run, and choose the Steam installer to complete the setup.
  </Step>
  <Step num="4" title="Launch the Game">
    Open Steam within your bottle, log into your account, locate the game in your library, and click install to start playing.
  </Step>
</Steps>

<Callout type="tip" title="Steam Cloud Saves">
Enable Steam Cloud in the game properties to sync your progress between your Mac and any other devices you use for gaming.
</Callout>

</Section>
