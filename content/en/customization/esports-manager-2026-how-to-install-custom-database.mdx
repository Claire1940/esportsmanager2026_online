<Section icon={Shield} title="Manual Installation Steps for Custom Databases">

If you are not using the Steam Workshop, you will need to perform a manual installation. This is common for "Total Conversion" mods that change more than just player names. Following the correct directory path is critical for the game to recognize the new data.

<Steps>
  <Step num="1" title="Download and Extract">
    Download the database ZIP or RAR file from a trusted source. Use a tool like 7-Zip or WinRAR to extract the contents to a temporary folder on your desktop.
  </Step>
  <Step num="2" title="Locate Game Directory">
    Navigate to your local game files. For most users, this is found at: `C:\Users\[YourName]\Documents\Esports Manager 2026\Databases`. If the folder does not exist, create it manually.
  </Step>
  <Step num="3" title="Transfer Files">
    Copy the extracted folder (containing the `.db` or `.json` files) into the `Databases` directory. Ensure you do not have nested folders (e.g., `Databases/ModName/ModName/files`).
  </Step>
  <Step num="4" title="Verify File Structure">
    Open the newly moved folder. You should see a file named `config.json` or `meta.data`. This file tells the game how to load the specific assets.
  </Step>
</Steps>

| Installation Step | Common Error | Solution |
| :--- | :--- | :--- |
| **Extraction** | Corrupt ZIP file | Re-download with stable internet |
| **Pathing** | Wrong folder selected | Use the Documents path, not Program Files |
| **Naming** | Special characters in folder | Rename folder to simple English text |
| **Permissions** | Read-only folder access | Run your file explorer as Administrator |

<Callout type="success" title="Success">
Once the files are in the correct directory, they will appear in the "Database Selection" dropdown menu when you click "New Game" in the main menu.
</Callout>

</Section>

Simply placing the files in the folder is not enough; you must explicitly tell **Esports Manager 2026** to use the custom data instead of the default 2026 roster.

When you launch the game, follow these specific settings to ensure the custom database initializes correctly:

1.  Click on **New Career**.
2.  In the **Database Selection** screen, click the dropdown menu.
3.  Select your custom database name (e.g., "Global Realism Pack v1.2").
4.  Wait for the "Data Validation" progress bar to complete.
5.  Select the leagues you wish to load.
