<Section icon={Map} title="Пошаговое руководство по установке">

Чтобы запустить игру на вашем Mac, необходимо настроить среду совместимости. Использование Whisky — один из самых эффективных методов с открытым исходным кодом, доступных для пользователей macOS. Он использует Apple Game Porting Toolkit для трансляции вызовов DirectX в Metal, что позволяет игре работать плавно.

Следуйте этим шагам, чтобы настроить среду и запустить игру.

<Steps>
  <Step num="1" title="Загрузите и установите Whisky">
    Скачайте последнюю версию Whisky для macOS. Перетащите приложение в папку «Программы» и откройте его для завершения первоначальной настройки.
  </Step>
  <Step num="2" title="Создайте новую «бутылку» (Bottle)">
    Нажмите кнопку создания бутылки. Выберите Windows 10 в качестве версии операционной системы и дайте бутылке имя. В этой среде будет размещена ваша установка Steam.
  </Step>
  <Step num="3" title="Установите Steam внутри бутылки">
    Загрузите установщик Steam для Windows (.msi или .exe). В Whisky выберите свою бутылку, нажмите «Запустить» (Run) и выберите установщик Steam для завершения настройки.
  </Step>
  <Step num="4" title="Запустите игру">
    Откройте Steam внутри вашей бутылки, войдите в свой аккаунт, найдите игру в библиотеке и нажмите «Установить», чтобы начать играть.
  </Step>
</Steps>

<Callout type="tip" title="Облачные сохранения Steam">
Включите Steam Cloud в свойствах игры, чтобы синхронизировать прогресс между вашим Mac и любыми другими устройствами.
</Callout>

</Section>
