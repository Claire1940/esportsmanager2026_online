<Section icon={Shield} title="Шаги по ручной установке пользовательских баз данных">

Если вы не используете Мастерскую Steam, вам потребуется выполнить ручную установку. Это часто встречается для модов «Total Conversion», которые меняют не только имена игроков. Следование правильному пути к директории критически важно для того, чтобы игра распознала новые данные.

<Steps>
  <Step num="1" title="Скачивание и извлечение">
    Скачайте ZIP или RAR файл базы данных из проверенного источника. Используйте такой инструмент, как 7-Zip или WinRAR, чтобы извлечь содержимое во временную папку на рабочем столе.
  </Step>
  <Step num="2" title="Поиск игровой директории">
    Перейдите к локальным файлам игры. Для большинства пользователей они находятся по адресу: `C:\Users\[ВашеИмя]\Documents\Esports Manager 2026\Databases`. Если папка не существует, создайте ее вручную.
  </Step>
  <Step num="3" title="Перенос файлов">
    Скопируйте извлеченную папку (содержащую файлы `.db` или `.json`) в директорию `Databases`. Убедитесь, что у вас нет вложенных папок (например, `Databases/ModName/ModName/files`).
  </Step>
  <Step num="4" title="Проверка структуры файлов">
    Откройте только что перемещенную папку. Вы должны увидеть файл с именем `config.json` или `meta.data`. Этот файл сообщает игре, как загружать конкретные ресурсы.
  </Step>
</Steps>

| Шаг установки | Распространенная ошибка | Решение |
| :--- | :--- | :--- |
| **Извлечение** | Поврежденный ZIP-файл | Повторно скачать при стабильном интернете |
| **Путь** | Выбрана неверная папка | Используйте путь в «Документах», а не в Program Files |
| **Именование** | Спецсимволы в названии папки | Переименуйте папку, используя простую латиницу |
| **Права доступа** | Доступ к папке только для чтения | Запустите проводник от имени администратора |

<Callout type="success" title="Успех">
Как только файлы окажутся в нужной директории, они появятся в выпадающем меню «Выбор базы данных» (Database Selection) при нажатии кнопки «Новая игра» в главном меню.
</Callout>

<Section icon={Sparkles} title="Активация базы данных в игре">

Простого размещения файлов в папке недостаточно; вы должны явно указать **Esports Manager 2026** использовать пользовательские данные вместо стандартного состава 2026 года.

При запуске игры следуйте этим настройкам, чтобы обеспечить корректную инициализацию пользовательской базы данных:

1.  Нажмите **Новая карьера** (New Career).
2.  На экране **Выбора базы данных** (Database Selection) нажмите на выпадающее меню.
3.  Выберите название вашей пользовательской базы данных (например, "Global Realism Pack v1.2").
4.  Дождитесь завершения полосы прогресса «Валидация данных» (Data Validation).
5.  Выберите лиги, которые вы хотите загрузить.

| Функция | База данных по умолчанию | Пользовательская база данных |
| :--- | :--- | :--- |
| **Имена игроков** | Вымышленные/Случайные | Реальные профессионалы |
| **Логотипы команд** | Стандартные фигуры | Официальный брендинг команд |
| **Призовые фонды** | Оценочные | Реальные цифры 2026 года |
| **Названия спонсоров** | Выдуманные бренды | Мировые технологические/энергетические бренды |
| **Атрибуты персонала** | Сбалансированные/Стандартные | На основе реальных показателей |

<Callout type="info" title="Важное примечание">
Некоторые базы данных являются «тяжелыми» и могут увеличить время начальной загрузки игры. Если кажется, что игра зависла на 88%, дайте ей несколько минут для обработки тысяч новых записей об игроках.
</Callout>

Если вы заметили, что имена верны, но логотипы отсутствуют, возможно, вам потребуется установить отдельный «Graphics Pack». Базы данных часто отвечают за текстовые данные, в то время как логотипы и лица игроков хранятся в отдельной папке `Graphics`.

| Тип ресурса | Имя папки | Расширение файла |
| :--- | :--- | :--- |
| **Составы/Статистика** | `Databases` | .db / .json |
| **Логотипы/Лица** | `Graphics` | .png / .fmf |
| **Скины/Интерфейс** | `Interface` | .xml |
| **Языковые пакеты** | `Languages` | .txt / .xml |

<Callout type="tip" title="Совет по оптимизации">
Для лучшей производительности загружайте только те лиги, в которых вы планируете играть. Загрузка каждой региональной лиги из массивной пользовательской базы данных может значительно замедлить скорость обработки хода при нажатии кнопки «Продолжить».
