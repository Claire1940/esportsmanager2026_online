<Section icon={Shield} title="Pasos para la instalación manual de bases de datos personalizadas">

Si no estás usando Steam Workshop, necesitarás realizar una instalación manual. Esto es común para mods de \"Conversión Total\" que cambian más que solo los nombres de los jugadores. Seguir la ruta de directorio correcta es fundamental para que el juego reconozca los nuevos datos.

<Steps>
  <Step num="1" title="Descargar y Extraer">
    Descarga el archivo ZIP o RAR de la base de datos desde una fuente confiable. Usa una herramienta como 7-Zip o WinRAR para extraer el contenido en una carpeta temporal en tu escritorio.
  </Step>
  <Step num="2" title="Localizar el Directorio del Juego">
    Navega a tus archivos locales del juego. Para la mayoría de los usuarios, se encuentra en: `C:\Usuarios\[TuNombre]\Documentos\Esports Manager 2026\Databases`. Si la carpeta no existe, créala manualmente.
  </Step>
  <Step num="3" title="Transferir Archivos">
    Copia la carpeta extraída (que contiene los archivos `.db` o `.json`) en el directorio `Databases`. Asegúrate de no tener carpetas anidadas (por ejemplo, `Databases/NombreMod/NombreMod/archivos`).
  </Step>
  <Step num="4" title="Verificar la Estructura de Archivos">
    Abre la carpeta que acabas de mover. Deberías ver un archivo llamado `config.json` o `meta.data`. Este archivo le indica al juego cómo cargar los recursos específicos.
  </Step>
</Steps>

| Paso de Instalación | Error Común | Solución |
| :--- | :--- | :--- |
| **Extracción** | Archivo ZIP corrupto | Volver a descargar con una conexión estable |
| **Ruta** | Carpeta incorrecta seleccionada | Usa la ruta de Documentos, no la de Archivos de programa |
| **Nombre** | Caracteres especiales en la carpeta | Renombra la carpeta con texto simple en inglés |
| **Permisos** | Acceso de carpeta de solo lectura | Ejecuta tu explorador de archivos como Administrador |

<Callout type="success" title="Éxito">
Una vez que los archivos estén en el directorio correcto, aparecerán en el menú desplegable \"Selección de Base de Datos\" al hacer clic en \"Nueva Partida\" en el menú principal.
</Callout>

</Section>

<Section icon={Sparkles} title="Activación de la base de datos en el juego">

Colocar los archivos en la carpeta no es suficiente; debes indicarle explícitamente a **Esports Manager 2026** que utilice los datos personalizados en lugar de la plantilla predeterminada de 2026.

Cuando inicies el juego, sigue estos pasos específicos para asegurarte de que la base de datos personalizada se inicialice correctamente:

1. Haz clic en **Nueva Carrera**.
2. In la pantalla de **Selección de Base de Datos**, haz clic en el menú desplegable.
3. Selecciona el nombre de tu base de datos personalizada (por ejemplo, \"Global Realism Pack v1.2\").
4. Espera a que se complete la barra de progreso de \"Validación de Datos\".
5. Selecciona las ligas que deseas cargar.

| Característica | Base de Datos Predeterminada | Base de Datos Personalizada |
| :--- | :--- | :--- |
| **Nombres de Jugadores** | Ficticios/Aleatorios | Profesionales Reales |
| **Logotipos de Equipos** | Formas Genéricas | Imagen Oficial del Equipo |
| **Premios de Torneos** | Estimados | Cifras Reales de 2026 |
| **Nombres de Patrocinadores** | Marcas Inventadas | Marcas Globales de Tecnología/Energía |
| **Atributos del Personal** | Equilibrados/Estándar | Basados en el Rendimiento Real |

<Callout type="info" title="Nota Importante">
Algunas bases de datos son \"Grandes\" y pueden aumentar el tiempo de carga inicial de tu juego. Si el juego parece congelarse en el 88%, dale unos minutos para procesar las miles de nuevas entradas de jugadores.
</Callout>

</Section>
